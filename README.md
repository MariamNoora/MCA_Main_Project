# ASD Detection Project

ML-based project for ASD detection using brain imaging data with a backend and frontend interface.

## Run

Backend:

```bash
pip install -r requirements.txt
python app.py
```

Frontend:

```bash
cd frontend
npm install
npm run dev
```

## Note

Large files (`.nii.gz`, `Samples/`, `ExBox17/`, `.ckpt`) are excluded from this repo.
